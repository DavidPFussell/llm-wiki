# Workflow Optimization for LLM Agents

## Definition

Workflow optimization studies how to design, search, adapt, and evaluate the computational structure of LLM-agent systems, including planning components, tool-use dependencies, memory updates, and verifier loops.

## Current Synthesis

- The survey on static templates versus dynamic runtime graphs provides the clearest taxonomy in this collection.
- Harness papers can be read as a special case of workflow optimization focused on executable context-management code.
- Terminal coding-agent work adds practical lessons about context engineering and safe operational scaffolding.
