# Causal Representation Learning

## Definition

Causal representation learning studies how models can discover latent variables and representations that reflect underlying causal structure rather than only predictive association.

## Current Synthesis

- It is one of the cleanest bridges between modern representation learning and causal reasoning.
- This concept is especially relevant if the wiki later tries to connect mechanism, abstraction, and world-modeling.
