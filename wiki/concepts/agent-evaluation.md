# Agent Evaluation

## Definition

Agent evaluation studies how to assess the capabilities, reliability, safety, cost, and robustness of LLM-based agents in realistic multi-step settings.

## Current Synthesis

- Evaluation is shifting from narrow task metrics toward more realistic and dynamic benchmarks.
- Validator quality is itself a serious problem, especially when LLMs are used as judges.
- Cost-efficiency, robustness, and safety remain underdeveloped compared with capability measurement.
