# Agent Memory

## Definition

Agent memory refers to the mechanisms by which agents store, retrieve, share, compress, and operationalize prior information or execution experience across turns, tasks, or collaborating agents.

## Current Synthesis

- `Trajectory-Informed Memory Generation` treats memory as structured learning extracted from execution traces.
- `Multi-Agent Memory from a Computer Architecture Perspective` treats memory as a systems problem involving hierarchy, sharing, and consistency.
- `SAGE` combines reflective reasoning with memory optimization for longer-horizon behavior.
