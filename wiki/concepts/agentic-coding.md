# Agentic Coding

## Definition

Agentic coding studies systems that perform software engineering tasks through multi-step planning, tool use, debugging, testing, and codebase interaction.

## Current Synthesis

- A recurring theme is that long-horizon coding requires more than code generation alone.
- Multi-agent coordination, testing, debugging, and repository navigation are central recurring ingredients.
- Full-stack and team-based software engineering are emerging as more demanding evaluation settings than toy code generation tasks.
