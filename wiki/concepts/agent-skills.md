# Agent Skills

## Definition

Agent skills are reusable procedural assets, such as workflows, code, or structured tool-use patterns, that can be accumulated, transferred, evaluated, and improved independently of base model weights.

## Current Synthesis

- `EvoSkill` focuses on automated discovery and refinement of skills through failure analysis.
- `SkillNet` focuses on large-scale infrastructure for creating, evaluating, and connecting skills.
- `Automating Skill Acquisition through Large-Scale Mining of Open-Source Agentic Repositories` focuses on extracting skills from external repositories into a standardized form.
