# Vector Retrieval and Approximate Search

## Definition

Vector retrieval and approximate search study how dense embeddings can be indexed and queried efficiently for nearest-neighbor retrieval at scale.

## Current Synthesis

- This is one of the main infrastructural layers underneath modern embedding-based retrieval and vector databases.
- It directly connects to RAG, semantic search, and any future local search engine work for the wiki.
