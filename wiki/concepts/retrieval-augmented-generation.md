# Retrieval-Augmented Generation

## Definition

Retrieval-augmented generation combines a language model with external evidence retrieval so generation can be conditioned on documents, records, or ranked knowledge at query time.

## Current Synthesis

- RAG is the dominant baseline for grounding language models on external corpora.
- The interesting frontier in this corpus is not basic RAG, but ranking, structuring, long-context integration, and agentic control around retrieval.
- This concept is especially relevant to the LLM Wiki project because it clarifies what persistent wiki compilation changes relative to standard RAG.
