# Graph Representation Learning

## Definition

Graph representation learning covers embeddings, neural message passing, and related methods for learning useful vector or structural representations of nodes, edges, paths, or whole graphs.

## Current Synthesis

- It is the main bridge between classical graph structure and modern neural learning.
- This concept will likely become a central parent page for embeddings, GNNs, and graph-scale learning systems.
