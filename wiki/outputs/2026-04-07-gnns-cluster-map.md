# 2026-04-07 GNNs Cluster Map

## Overview

The `GNNs` folder narrows the broader graph-learning area into a specific architectural family. It covers graph convolution, inductive classification, scaling variants, and applications of graph neural networks to text and temporal data.

## Main Threads

- **Core GNN framing**: [GNNs A Review of Methods and Applications](../sources/gnns-a-review-of-methods-and-applications.md) provides the broad family map.
- **GCN lineage**: [Semi-supervised Classification with GCNs](../sources/semi-supervised-classification-with-gcns.md), [GCNs for Text Classification](../sources/gcns-for-text-classification.md), and [FastGCN](../sources/fastgcn.md) form the core technical spine.
- **Text and hierarchy applications**: [Inductive Text Classification via GNNs](../sources/inductive-text-classification-via-gnns.md) and [Large-Scale Hierarchical Text Classification with Recursivels Regularized Deep GCNN](../sources/large-scale-hierarchical-text-classification-with-recursivels-regularized-deep-gcnn.md) connect GNNs back to document organization.
- **Temporal graph modeling**: [Spatio-Temporal Graph Convolutional Networks](../sources/spatio-temporal-graph-convolutional-networks-a-deep-learning-framework.md) broadens the cluster beyond text and static graphs.
