# 2026-04-07 Intent Cluster Map

## Overview

The `Intent` folder is compact and applied. It focuses on user-intent detection under limited labels, open-world classes, and sentence-encoder-heavy setups.

## Main Threads

- **Survey and framing**: [Intent Detection Survey](../sources/intent-detection-survey.md) gives the high-level map.
- **Encoder-centric intent models**: [ConveRT](../sources/convert.md) and [Efficient Intent Detection with Dual Sentence Encoders](../sources/efficient-intent-detection-with-dual-sentence-encoders.md) show the sentence-embedding route.
- **Low-resource and open-world settings**: [Few-shot Intent Detection](../sources/few-shot-intent-detection.md), [Unknown Intent Detection](../sources/unknown-intent-detection.md), and [Zero-shot User Intent Detection](../sources/zero-shot-user-intent-detection.md) define the robustness side.

## Current Read

- This cluster is a strong application layer for sentence embeddings.
- It also connects naturally to CX and sentiment where intent, tone, and service outcome are tightly coupled.
