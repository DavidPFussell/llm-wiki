# 2026-04-07 Sentiment Cluster Map

## Overview

The `Sentiment` folder is broader than sentiment alone. It includes sentiment analysis, politeness, impoliteness, stylistic transfer, and a few emotionally grounded semantic sources.

## Main Threads

- **Sentiment-analysis framing**: [Current Challenges and New Directions in Sentiment Analysis Research](../sources/current-challenges-and-new-directions-in-sentiment-analysis-research.md) sets the survey layer.
- **Politeness and interaction style**: [Computational Approach to Politeness](../sources/computational-approach-to-politeness.md), [Impoliteness](../sources/impoliteness.md), [Politeness Transfer](../sources/politeness-transfer.md), and [Polite Dialogue Generation](../sources/polite-dialogue-generation.md) define the interaction-style branch.
- **Embedding-based sentiment signals**: [Sentiment Classification using Document Embeddings Trained with Cosine Similarity](../sources/sentiment-classification-using-document-embeddings-trained-with-cosine-similarity.md) and [Learning Semantic Sentence Embeddings using Pair-Wise Discriminator](../sources/learning-semantic-sentence-embeddings-using-pair-wise-discriminator.md) connect the shelf back to sentence representation.

## Current Read

- This cluster is especially useful as a bridge between language representation and service interaction quality.
- It pairs naturally with CX and intent detection.
