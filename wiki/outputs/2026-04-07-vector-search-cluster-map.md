# 2026-04-07 Vector Search Cluster Map

## Overview

The `Vector Search` folder is compact and infrastructural. It focuses on the dense-retrieval layer that sits underneath embedding-based search, semantic retrieval, and many modern RAG systems.

## Main Threads

- **Foundations of dense retrieval**: [Foundations of Vector Retrieval](../sources/foundations-of-vector-retrieval.md) provides the high-level framing.
- **Approximate nearest-neighbor methods**: [Locality Sensitive Hashing](../sources/locality-sensitive-hashing.md) anchors the algorithmic side.
- **Practical retrieval systems**: [Vector Search with OpenAI Embeddings Lucene is All You Need](../sources/vector-search-with-openai-embeddings-lucene-is-all-you-need.md) ties theory back to modern systems practice.

## Current Read
