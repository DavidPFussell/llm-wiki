# 2026-04-07 Cross-Validation Cluster Map

## Overview

The `Cross-validation` folder is small but important. It sharpens the repo’s treatment of evaluation, model selection, and especially the harder problem of validation for unsupervised methods.

## Main Threads

- **Unsupervised validation**: [Cross-validation for unsupervised learning](../sources/cross-validation-for-unsupervised-learning.md) and [Estimating the number of clusters using cross-validation](../sources/estimating-the-number-of-clusters-using-cross-validation.md) are the core of the cluster.
- **Robustness and influential observations**: [Cross-validation_influential_observation](../sources/cross-validation-influential-observation.md) broadens the cluster toward sensitivity analysis.
- **Prediction-strength perspective**: [Cluster validation by prediction strength](../sources/cluster-validation-by-prediction-strength.md) provides a direct bridge into clustering quality assessment.

## Current Read

- This folder is less about a standalone domain than about giving the whole repo stronger evaluation discipline.
- It is especially relevant to the clustering cluster, where validation is often the weakest link.
