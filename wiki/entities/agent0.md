# Agent0

## Overview

Agent0 is a self-evolving agent framework built around co-evolution between a curriculum agent and an executor agent, with tool integration used to increase task complexity and learning pressure.

## Why It Matters

It is one of the clearest zero-data self-improvement systems in the collection and a useful counterpart to Darwin Godel Machine because it emphasizes curriculum generation rather than archive-based code evolution.
