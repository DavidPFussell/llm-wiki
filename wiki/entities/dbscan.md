# DBSCAN

## Overview

DBSCAN is a density-based clustering method that remains a core reference point for robust clustering, noise handling, and cluster discovery without specifying the number of clusters in advance.

## Why It Matters

It is a canonical anchor for practical clustering workflows and robustness discussions.
