# Titans

## Overview

Titans is a test-time memory architecture proposal focused on improving sequence modeling by learning to memorize during inference.

## Why It Matters

It is a useful anchor for the part of the corpus where reasoning quality and context use depend on test-time memory structure rather than only larger static context windows.
