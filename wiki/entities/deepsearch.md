# DeepSearch

## Overview

DeepSearch is a reinforcement-learning and search-oriented system that uses Monte Carlo Tree Search with verifiable rewards to improve reasoning or problem-solving behavior.

## Why It Matters

It is a strong anchor for the branch of RL-for-LLMs work that leans on explicit search and verifiable objectives.
