# GCN

## Overview

GCN refers to graph convolutional network architectures that became one of the canonical baselines for graph neural networks.

## Why It Matters

It is the strongest single entity-level anchor in the current `GNNs` cluster and a useful bridge back to graph representation learning more broadly.
