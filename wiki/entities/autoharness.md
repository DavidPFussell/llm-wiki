# AutoHarness

## Overview

AutoHarness is a system that automatically synthesizes executable code harnesses around LLM agents using environment feedback and iterative refinement.

## Why It Matters

It is a key entity in the harness-design thread because it treats the harness itself as a learnable artifact rather than fixed plumbing.
