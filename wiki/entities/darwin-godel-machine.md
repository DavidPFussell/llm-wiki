# Darwin Godel Machine

## Overview

The Darwin Godel Machine is a self-improving coding-agent system proposed by Jenny Zhang, Shengran Hu, Cong Lu, Robert Lange, and Jeff Clune. It combines iterative self-modification with open-ended evolutionary search over an archive of agent variants.

## Why It Matters

DGM is a useful anchor entity for this wiki because it bridges several strands of agent research:

- coding agents that modify and evaluate code
- recursive or self-improving systems
- open-ended exploration and evolutionary search
- safety-constrained experimentation through sandboxing and oversight

## Key Characteristics

- Empirical validation is used instead of formal proof of beneficial self-modification.
- The system samples from an archive of prior agents and branches into new variants.
- The reported improvements include better code-editing tools, long-context management, and peer-review mechanisms.
