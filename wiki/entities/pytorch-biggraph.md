# PyTorch-BigGraph

## Overview

PyTorch-BigGraph is a large-scale graph embedding system designed for industrial-scale representation learning over very large graphs.

## Why It Matters

It is a strong anchor for scale-oriented graph embedding infrastructure rather than only graph-learning theory.
