# Insight Agents

## Overview

Insight Agents is a conversational multi-agent data-insight system for e-commerce sellers built around a hierarchical manager-worker structure.

## Why It Matters

It is one of the strongest production-style systems in the `Agent` cluster and a useful anchor for the autonomous data science subthread.
