# MM-REACT

## Summary

MM-REACT uses prompting to connect language models with external multimodal experts rather than relying on end-to-end multimodal retraining alone. It is a useful bridge between tool use and multimodal reasoning.

## Key Contributions

- Prompt-based coordination of multimodal experts
- Demonstrates modular multimodal reasoning without full retraining
- Good bridge between tool use and multimodal agent systems
