# Near Synonym Choice Using 5-gram Language Model

## Summary

In this work, an unsupervised statistical method for automatic choice of near-synonyms is presented and compared to the stateof-the-art. We use a 5-gram language model built from the Google Web 1T data set. The proposed method works automatically, does not require any human-annotated knowledge resources (e.g., ontologies) and can be applied to diï¬€erent languages. Our evaluation experiments show that this method outperforms two previous methods on the same task.
