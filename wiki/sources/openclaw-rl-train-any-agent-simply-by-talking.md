# OpenClaw-RL: Train Any Agent Simply by Talking

## Summary

OpenClaw-RL argues that every agent interaction produces useful next-state signals and that these signals can support live online RL across conversational, terminal, GUI, SWE, and tool-call settings. It is especially relevant to questions about agents that improve simply through use.

## Key Contributions

- Unified treatment of diverse interaction traces as learning data
- Distinction between evaluative and directive next-state signals
- Asynchronous infrastructure for simultaneous serving, judging, and training
