# Agentic Context Engineering

## Summary

This paper introduces ACE, a framework for treating contexts as evolving playbooks that accumulate and refine strategy over time. It targets problems like brevity bias and context collapse, and positions context adaptation itself as a persistence layer for self-improving agents.

## Key Contributions

- Treats context as an evolving curated artifact rather than a static prompt
- Emphasizes modular generation, reflection, and curation
- Shows strong relevance to online agent memory and low-overhead self-improvement
