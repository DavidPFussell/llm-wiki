# Agent IR

## Summary

This position paper proposes agentic information retrieval as a new IR paradigm shaped by LLM agents. The core move is to replace fixed single-pass IR pipelines with recurrent reasoning-action loops that aim to move the user toward a desired information state.

## Key Contributions

- Reframes IR around agents rather than fixed retrieval architectures
- Expands task scope beyond ranking and retrieval alone
- Connects prompt engineering, RAG, fine-tuning, RL, and multi-agent methods to IR
