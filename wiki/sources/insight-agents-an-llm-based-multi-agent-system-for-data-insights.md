# Insight Agents: An LLM-Based Multi-Agent System for Data Insights

## Summary

Insight Agents is a hierarchical multi-agent data-insight system for e-commerce sellers. It combines a manager-worker architecture with routing, descriptive analytics, and diagnostic analysis under latency constraints.

## Key Contributions

- Practical plan-and-execute multi-agent architecture for data insight generation
- Lightweight manager routing for latency-sensitive deployment
- Strong product-style focus on coverage, accuracy, and latency together
