# Relational Deep Learning: Graph Representation Learning on Relational Databases

## Summary

This paper proposes a blueprint for end-to-end learning directly on relational databases by representing them as heterogeneous graphs. It is somewhat peripheral to the core LLM thread but relevant to data modeling and knowledge structure.

## Key Contributions

- End-to-end learning on relational databases without manual feature engineering
- Graph-based reframing of relational learning
- Useful bridge between databases, graphs, and learned representations
