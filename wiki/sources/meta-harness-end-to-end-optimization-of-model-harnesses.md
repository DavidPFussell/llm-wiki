# Meta-Harness: End-to-End Optimization of Model Harnesses

## Summary

Meta-Harness treats harness engineering as a search problem over executable code. Rather than manually designing what the model sees and remembers, it searches directly over harness implementations using prior traces and results.

## Key Contributions

- Outer-loop search over harness code
- Use of source code, scores, and traces from prior candidates
- Strong reported gains in classification, math reasoning, and agentic coding
