# Surfer 2

## Summary

Surfer 2 is a cross-platform computer-use agent operating from visual observations across web, desktop, and mobile environments. It emphasizes hierarchical context management, decoupled planning and execution, and self-verification with adaptive recovery.

## Key Contributions

- Unified visual agent across web, desktop, and mobile
- Decoupled planning and execution for long-horizon reliability
- Strong benchmark performance without task-specific fine-tuning
