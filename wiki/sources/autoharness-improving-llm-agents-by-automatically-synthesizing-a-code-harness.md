# AutoHarness: improving LLM agents by automatically synthesizing a code harness

## Summary

AutoHarness shows that a smaller model can synthesize an executable code harness that prevents illegal actions and can outperform larger frontier models in text-game environments. The core idea is that agent quality can improve dramatically when the surrounding harness is learned rather than hand-built.

## Key Contributions

- Automatic synthesis of code harnesses from environment feedback
- Elimination of illegal actions across many TextArena games
- Evidence that code-policy generation can outperform direct LLM decision making
