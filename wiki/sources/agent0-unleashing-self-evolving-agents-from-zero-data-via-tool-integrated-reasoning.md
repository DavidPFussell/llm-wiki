# Agent0: Unleashing Self-Evolving Agents from Zero Data via Tool-Integrated Reasoning

## Summary

Agent0 presents a fully autonomous self-evolution framework in which a curriculum agent and an executor agent co-evolve from the same base LLM. Tool use is built into the evolution loop so stronger execution produces harder frontier tasks, creating a self-reinforcing curriculum without external training data.

## Key Contributions

- Zero-data agent evolution rather than dependence on human-curated supervision
- Multi-step co-evolution between curriculum generation and execution
- Tool-integrated reasoning as a driver of richer task complexity
