# The Auton Agentic AI Framework

## Summary

Auton is an architecture paper that emphasizes the mismatch between stochastic LLM outputs and deterministic backend systems. Its central proposal is a separation between a cognitive blueprint and a runtime engine, with supporting ideas around memory, safety constraints, and execution optimization.

## Key Contributions

- Separation of cognitive specification from runtime execution
- Formal architecture for governance, portability, and safety
- Broad systems framing that includes memory, RL, and latency optimization
