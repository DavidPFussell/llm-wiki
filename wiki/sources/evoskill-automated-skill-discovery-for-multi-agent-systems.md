# EvoSkill: Automated Skill Discovery for Multi-Agent Systems

## Summary

EvoSkill proposes a self-evolving framework that discovers and refines reusable agent skills through iterative failure analysis. Instead of optimizing tightly coupled prompts or code fragments, it focuses on more transferable skill-level artifacts.

## Key Contributions

- Skill discovery from execution failures
- Pareto-based retention of useful skills
- Evidence for cross-task transfer of evolved skills
