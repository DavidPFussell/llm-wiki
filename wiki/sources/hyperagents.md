# Hyperagents

## Summary

Hyperagents extends the self-improving-agent line by integrating task agents and meta-agents into a single editable program. The framing emphasizes self-reference and metacognitive modification rather than a fixed outer-loop controller.

## Key Contributions

- Unified task-level and meta-level agent program
- Stronger self-referential framing for agent improvement
- Extends open-ended self-improvement ideas beyond a single coding benchmark setup
