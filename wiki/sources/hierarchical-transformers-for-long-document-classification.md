# Hierarchical Transformers for Long Document Classification

## Summary

This paper extends BERT-style modeling to long documents by segmenting the input and then modeling interactions between segment representations. It is a useful precursor to later long-context and hierarchical transformer work.

## Key Contributions

- Hierarchical extension of BERT for long-document classification
- Practical handling of sequences far beyond base Transformer limits
- Relevant to long-context document processing workflows
