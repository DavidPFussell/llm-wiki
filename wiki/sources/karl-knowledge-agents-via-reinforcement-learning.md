# KARL: Knowledge Agents via Reinforcement Learning

## Summary

KARL trains enterprise search agents through reinforcement learning and synthetic data generation, with a strong emphasis on heterogeneous search behaviors and multi-task generalization. It is one of the more application-oriented RL papers in the folder.

## Key Contributions

- KARLBench for diverse enterprise search behaviors
- Synthetic data generation pipeline for hard-to-verify search tasks
- Large-batch off-policy RL for grounded knowledge-agent behavior
