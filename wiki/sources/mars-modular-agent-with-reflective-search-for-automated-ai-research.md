# MARS: Modular Agent with Reflective Search for Automated AI Research

## Summary

MARS targets automated AI research with three main ideas: budget-aware planning, modular construction, and comparative reflective memory. It stands out by treating research as a resource-constrained search problem rather than a generic coding task.

## Key Contributions

- Budget-aware planning using cost-constrained search
- Modular construction for research repositories
- Comparative reflective memory for cross-branch learning
