# Tool Learning with Large Language Models: A Survey

## Summary

This survey organizes the tool-learning literature for LLMs around why tool learning matters and how it is implemented. It provides a useful taxonomy of task planning, tool selection, tool calling, and response generation.

## Key Contributions

- Broad survey of tool learning with LLMs
- Workflow-oriented taxonomy across key tool-use stages
- Useful benchmark and evaluation overview for tool-learning systems
