# From Static Templates to Dynamic Runtime Graphs: A Survey of Workflow Optimization for LLM Agents

## Summary

This survey organizes LLM-agent workflows as agentic computation graphs and distinguishes static scaffolds from dynamic run-specific structures. It is one of the clearest taxonomy papers in the folder and likely deserves to become a backbone reference for the wiki.

## Key Contributions

- Unifying abstraction of agentic computation graphs
- Taxonomy based on when workflow structure is determined
- Structure-aware evaluation perspective beyond downstream task scores
