# SAGE: Self-evolving Agents with Reflective and Memory-augmented Abilities

## Summary

SAGE proposes a framework of collaborative agents that combine iterative feedback, reflective reasoning, and memory optimization. Compared with some of the newer arXiv papers in the folder, it reads as an earlier structured attempt to improve long-horizon agent behavior through reflection and memory discipline.

## Key Contributions

- Multi-agent setup with assistant and checker roles
- Reflection and iterative feedback loops
- Memory optimization inspired by forgetting-curve ideas
