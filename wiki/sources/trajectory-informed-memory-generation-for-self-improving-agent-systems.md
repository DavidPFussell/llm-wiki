# Trajectory-Informed Memory Generation for Self-Improving Agent Systems

## Summary

This paper proposes a framework that extracts structured learnings from execution trajectories and retrieves them as contextual guidance in future tasks. It is a strong candidate for the memory-learning backbone of the collection because it focuses on provenance-rich, actionable memory rather than generic conversational storage.

## Key Contributions

- Semantic analysis of execution trajectories
- Attribution of failures, recoveries, and inefficiencies
- Retrieval of strategy, recovery, and optimization guidance tailored to new tasks
