# Who Validates the Validators?

## Summary

This paper studies the problem of aligning LLM-assisted evaluation with human preferences. It is directly relevant to agent evaluation because it treats the evaluator itself as an object that requires validation, iteration, and human-in-the-loop correction.

## Key Contributions

- Mixed-initiative framework for aligning evaluator behavior with human preferences
- Identifies criteria drift as a central evaluation problem
- Useful warning against overly trusting automatic LLM judges
