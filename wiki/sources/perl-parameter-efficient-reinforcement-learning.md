# PERL: Parameter Efficient Reinforcement Learning

## Summary

PERL studies RLHF under a parameter-efficient fine-tuning setup using LoRA. It is useful as a bridge between alignment training and efficient adaptation methods.

## Key Contributions

- Parameter-efficient RLHF with LoRA
- Comparable performance to conventional RLHF with lower cost
- Useful reference for efficient alignment workflows
