# Building AI Coding Agents for the Terminal: Scaffolding, Harness, Context Engineering, and Lessons Learned

## Summary

This paper presents OPENDEV, a terminal-native coding agent, and focuses on the systems lessons required for long-horizon autonomous software engineering. Its emphasis is practical: scaffolding, safety controls, routing, memory, and context compaction matter as much as the base model.

## Key Contributions

- Terminal-first agent architecture for software engineering
- Dual-agent planning and execution split
- Context-efficiency and memory mechanisms to prevent degradation over long runs
