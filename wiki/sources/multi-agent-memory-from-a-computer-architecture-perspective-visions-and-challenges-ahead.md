# Multi-Agent Memory from a Computer Architecture Perspective: Visions and Challenges Ahead

## Summary

This position paper argues that multi-agent memory should be treated as a computer architecture problem involving hierarchy, bandwidth, sharing, and consistency. It offers a systems lens that complements more algorithmic memory-generation work.

## Key Contributions

- Distinction between shared and distributed memory setups
- Proposal of a layered memory hierarchy for agent systems
- Argument that memory consistency is a core unsolved challenge
