# Obsidian Setup

Recommended setup for using this repo as an Obsidian vault:

- Open the repo root as the vault.
- Save clipped web articles into `raw/sources/`.
- Save downloaded attachments into `raw/assets/`.
- Use graph view to inspect link structure, hubs, and orphans.
- Optionally install:
  - Marp for slide decks
  - Dataview for frontmatter-driven tables and lists

If you use Obsidian Web Clipper, pair it with local attachment downloads so multimodal agents can inspect referenced images directly from disk.
