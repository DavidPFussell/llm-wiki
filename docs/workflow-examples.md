# Workflow Examples

## Ingest Example

`Ingest raw/sources/new-paper.md. Create or update the relevant source, entity, and concept pages, then refresh index.md and append a log entry.`

## Query Example

`Using the wiki, compare the strongest arguments for and against this approach. Write the result to wiki/outputs/ as a markdown page with citations.`

## Lint Example

`Run a lint pass over wiki/concepts/ and wiki/entities/. Look for duplicates, missing cross-links, and stale claims. Write findings to wiki/maintenance/ and append a log entry.`
